# Warehouse-Inventory-Dashboard
## Project Objective
Designed an interactive Warehouse Inventory Management Dashboard to monitor stock levels, inventory value, supplier contribution, and reorder status across multiple warehouses. The dashboard enables quick decision-making by providing real-time inventory insights through dynamic visuals and KPIs.

## Dataset Used
- <a href="https://github.com/devrajwrk-coder/Warehouse-Inventory-Dashboard/blob/main/Warehouse_Inventory_Dashboard.xlsx">Dataset</a>

## Questions (KPIs)
- What is the total inventory value and stock quantity?
- Which warehouse holds the highest stock?
- How many items require reordering?
- Which category and supplier contribute the most to inventory value?
- What are the top 10 products by stock quantity?
